# BookIQ AI Dashboard
A futuristic book intelligence dashboard featuring AI integration and web scraping simulations.

## Features
- **AI Assistant:** Interface for interacting with LLMs for book analysis.
- **Web Scraper:** UI for automated data extraction simulations.
- **Real-time Stats:** Visual representation of book data metrics.
- **Responsive Design:** Fully functional on mobile and desktop.

## Technologies Used
- HTML5 / CSS3 (Modern Flexbox/Grid)
- JavaScript (ES6+)
- Lucide Icons
